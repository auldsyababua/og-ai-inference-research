# Scripts Directory

This directory contains utility scripts for the project.

---

## Directory Structure

- **`archive/`** - Scripts that may be reused but are not actively used
  - `setup-gitlab.sh` - GitLab repository setup script (may be reused for new repos)
  - `reorganize.sh` - Project reorganization script (historical)
  - `GITLAB-PUSH-ISSUE.md` - GitLab push troubleshooting notes

- **`tmp/`** - Temporary scripts for one-time tasks
  - `FIX_GITLAB_PUSH.sh` - Temporary script for GitLab push troubleshooting

---

## Usage

Scripts in `archive/` may be referenced for similar tasks but are not part of the active workflow.

Scripts in `tmp/` are temporary and can be deleted after the task is complete.
